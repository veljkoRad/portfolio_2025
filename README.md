## Veljko Portfolio 2025

Live: https://veljko-dev-2025.netlify.app  

My personal frontend portfolio. It shows the type of work I actually do: landing pages, React projects and production HTML email templates.

---

## Tech stack

- React (Create React App)
- Material UI (MUI 7) + Emotion
- Framer Motion
- react-hook-form
- react-circular-progressbar

---

## What’s inside

- **Hero** – intro, dark/light theme toggle, “Download CV” and “My Work” buttons.
- **About** – short description + tabs for about/experience/education with animated skill circles.
- **Skills** – cards describing what I build (landing pages, blogs, emails, WordPress).
- **Projects** – tabs for React, landing pages, email templates (all data in `projectData.js`).
- **Contact** – Netlify-ready contact form using `react-hook-form`.
- **Footer** – social links (LinkedIn, GitHub, Facebook, Instagram).

---

npm install
