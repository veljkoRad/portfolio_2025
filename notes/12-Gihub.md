## GIT Local

1. `git status`
  - If says "not a git repository I need to instal it
2. `git init`
3. ` git add .`
4. `git commit -m 'first commit'`
5. `git branch` to check my branch name

## GIT Remote

1. `git remove -v`
  - If nothing appears, not installed

2. `git remote add portfolio git@github.com:veljkoRad/portfolio_2025.git`
3. `git push portfolio master`

### When pull from another pc:

1. `git init`

2. `git remote add portfolio git@github.com:veljkoRad/portfolio_2025.git`

3. `git pull origin master`

4. `npm install`
