## Arrow function with explicit return

```
const f =() => {
  return 
}
```

## Arrow function with implicit return. Implicit return is used only in arrow function

`const f =() =>  5` 

## React arrow function component explicit return

```
const f =() => {
  return  <div>A</div>
}

  or multiline 

const f =() => {
  return (
    <section>
      <h1>Hello</h1>
      <p>World</p>
    </section>
  )
}

```

 )
